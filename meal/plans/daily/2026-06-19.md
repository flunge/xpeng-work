## 🌅 早餐

### 吐司小披萨 + 苹果胡萝卜汁

**用时：** 20分钟（前一晚10分钟 + 早上10分钟）

**工具：** 空气炸锅、破壁机

### 用料

**吐司小披萨：**

- 吐司片 4片
- 番茄酱 3大勺
- 马苏里拉芝士碎 适量
- 玉米粒 小半碗
- 青豆 小半碗
- 鸡胸肉 半块（约80g，煮熟切小丁，代替火腿丁）
- 小番茄 4颗（切片）
- 鸡蛋 2个（每个吐司打半个鸡蛋在芝士下）

**苹果胡萝卜汁：**

- 苹果 2个（去皮去核切块）
- 胡萝卜 1根（去皮切小块）
- 蜂蜜 少许（可选）

### 🕐 前一晚准备

- [ ] 玉米粒、青豆焯水1分钟，沥干冷藏
- [ ] 鸡胸肉冷水下锅煮8分钟，焖5分钟至熟透，捞出切小丁，放入保鲜盒冷藏
- [ ] 小番茄切片，放入保鲜盒冷藏
- [ ] 苹果和胡萝卜切块，分开放保鲜盒冷藏（苹果加柠檬汁防氧化）

### 🌤️ 早上操作（≤30分钟）

1. 吐司片摆在铺了油纸的烤盘上，涂一层番茄酱
2. 撒上玉米粒、青豆、鸡肉丁，铺上小番茄片
3. 鸡蛋打散，每个吐司上淋少许蛋液，再撒上马苏里拉芝士碎
4. 空气炸锅180℃预热3分钟，放入吐司烤6-8分钟至芝士融化金黄、蛋液凝固
5. 同时将苹果块、胡萝卜块和200ml水放入破壁机打成果蔬汁（约1分钟）
6. 披萨切块，搭配果蔬汁上桌

💡 这个早餐孩子可以参与放配料（他们超爱自己做披萨）。芝士拉丝的效果会让他们很兴奋。果蔬汁现打现喝，营养不流失。

### 🎉 端午节特别：自己包蜜枣粽子

**用时：** 前一晚约3小时（泡米+包+煮）+ 早上10分钟回热

**工具：** 炖锅、蒸锅、电饭煲

### 用料

**蜜枣粽子（约12-15个）：**

- 圆糯米 500g（黏性足，比长糯米更适合包甜粽）
- 蜜枣 15颗（给孩子吃建议提前去核，防噎）
- 干粽叶 30片（一个粽子用2片，提前泡软煮过）
- 棉线（粽绳）1卷（食品级棉线）
- 葡萄干 1小把（可选，和蜜枣一起包增加风味）

### 🕐 前一晚准备（端午亲子活动）

- [ ] 圆糯米淘洗2-3遍，清水浸泡4小时以上（过夜更省事），泡好后沥干水分
- [ ] 干粽叶冷水泡软（2小时以上），下锅加水煮5-10分钟杀菌变软，捞出洗净沥干；棉线一起入锅煮软备用
- [ ] 蜜枣温水泡10分钟洗净，给孩子吃的提前去核
- [ ] 包粽子：取2片粽叶叠放折成漏斗状，先放一层糯米→中间放2颗蜜枣→再盖一层糯米压实，把粽叶折回收紧成三角，用棉线扎紧（全家一起包，孩子可帮忙放枣、递叶子）
- [ ] 煮粽子：粽子冷水下炖锅，水没过粽子，大火烧开转小火煮2-2.5小时（中途加热水保持没过）；也可用电饭煲煮2小时
- [ ] 煮好关火再焖30分钟，捞出沥干（吃不完冷藏保存）

### 🌤️ 早上操作（≤30分钟）

1. 前一晚煮好的粽子放蒸锅，上汽后蒸8-10分钟回热（或电饭煲加热）
2. 剪开棉线，剥开粽叶装盘，喜欢甜的可以蘸一点点白糖
3. 搭配牛奶或果蔬汁，孩子1-2个即可，糯米不易消化别贪多

💡 自己包粽子是端午很好的亲子活动，孩子放枣、收叶会很有成就感。蜜枣务必去核防噎；糯米偏黏不好消化，小朋友1-2个为宜。煮的时间长，建议前一晚包好煮好，早上只蒸热。

## ☀️ 午餐

### 南瓜鸡肉焖饭 + 黄瓜蛋汤

**南瓜鸡肉焖饭：**

- 大米 2杯
- 去骨鸡腿肉 300g（切小块）
- 南瓜 200g（去皮切小丁）
- 干香菇 4朵（温水泡发后切丁）
- 胡萝卜 半根（切小丁）
- 宝宝酱油 1大勺
- 蚝油 1小勺
- 姜末 少许
- 葱花 少许

**黄瓜蛋汤：**

- 黄瓜 1根（切片）
- 鸡蛋 2个
- 葱花 少许
- 盐 少许
- 香油 几滴

### 做法

- 鸡腿肉切小块，加酱油、蚝油、姜末腌制15分钟
- 大米淘洗放入电饭煲，正常煮饭水量 🔧电饭煲
- 将南瓜丁、胡萝卜丁、香菇丁、腌好的鸡肉块铺在米上 🔧电饭煲
- 按下煮饭键，正常煮饭 🔧电饭煲
- 饭好后打开盖子，撒葱花，将南瓜和饭拌匀
- 汤锅加水烧开，放入黄瓜片煮1分钟 🔧玻璃锅
- 淋入打散的蛋液，加盐调味 🔧玻璃锅
- 撒葱花，淋香油
- 南瓜鸡肉焖饭和黄瓜蛋汤一起上桌

### ➕ 加菜：清炒荷兰豆胡萝卜

**清炒荷兰豆胡萝卜：**

- 荷兰豆 200g（撕去筋）
- 胡萝卜 半根（切薄片）
- 蒜 2瓣（切末）
- 盐 少许

**做法：**

- 荷兰豆和胡萝卜片入开水焯1分钟，捞出沥干 🔧玻璃锅
- 炒锅倒少许油，下蒜末炒香，放入荷兰豆和胡萝卜快炒，加少许盐调味即可 🔧炒锅

## 🌙 晚餐

### 玉米蒸肉饼 + 蒜蓉西兰花

**玉米蒸肉饼：**

- 猪梅花肉 300g（剁成肉末或用料理机打碎）
- 甜玉米粒 小半碗
- 胡萝卜 半根（切碎末）
- 宝宝酱油 1大勺
- 淀粉 1大勺
- 姜末 少许
- 葱花 少许

**蒜蓉西兰花：**

- 西兰花 1颗（切小朵）
- 蒜 3瓣（切末）
- 盐 少许

**主食：**

- 大米 2杯

### 做法

- 肉末中加入玉米粒、胡萝卜碎、酱油、淀粉、姜末，顺时针搅拌上劲
- 将肉馅平铺在深盘中，厚度约1.5cm
- 大米淘洗入电饭煲，加适量水按下煮饭键 🔧电饭煲
- 蒸锅水烧开，放入肉饼盘，中火蒸约25分钟至熟透 🔧蒸锅
- 炒锅烧水，西兰花焯水1分钟，捞出沥干 🔧炒锅
- 炒锅倒少许油，下蒜末炒香 🔧炒锅
- 放入西兰花快速翻炒，加少许盐调味即可 🔧炒锅
- 肉饼蒸好后取出，撒葱花，切块装盘

### ➕ 加菜：凉拌豆腐丝

**凉拌豆腐丝：**

- 豆腐丝 200g
- 黄瓜 半根（切丝）
- 胡萝卜 少许（切丝）
- 蒜 2瓣（切末）
- 宝宝酱油 1大勺
- 香油 几滴

**做法：**

- 玻璃锅烧开水，豆腐丝焯1分钟去豆腥，捞出过凉沥干 🔧玻璃锅
- 豆腐丝加黄瓜丝、胡萝卜丝、蒜末、酱油、香油拌匀即可
