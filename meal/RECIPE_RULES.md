# 家庭食谱规则 / 记忆（沉淀自与用户的对话）

> 这是食谱系统的"宪法"。新增/修改食谱、跑 `generate_month.py` 前都应遵循。配置项见 `config/family.yaml`。

## 一、家庭与份量
- **4 人**：2 成人 + 2 儿童（**7 岁、5 岁男孩，正在长身体**）。
- 食谱 `servings: 4`，分量按"2 大 + 2 小"推断；两小孩合计约 1 个成人饭量，但**必须保证优质蛋白**（长身高）。

## 二、口味（很重要：孩子爱吃才有意义！）
- **不吃辣**；**少油少盐**。
- **优先孩子爱吃的口味**（营养均衡前提下）：酸甜、番茄味、芝士/奶香、照烧/微甜咸香、蜂蜜烤香、玉米/南瓜/紫薯自然甜、肉松/海苔鲜香。
- 做不健康但没意义——口味与营养并重。

## 三、烹饪方式约束（健康）
- **不用油锅油炸**（油炸不健康）。**例外**：炸葱油这类小型熬油可以。
- **空气炸锅常用、首选**——用它的少油"炸/烤"替代油炸（如脆皮、薯角、烤鸡翅、烤吐司）。
- **没有微波炉**——任何步骤不得依赖微波。
- 可用厨具：破壁机 / 空气炸锅 / 不粘煎锅 / 炖锅 / 玻璃锅 / 炒锅 / 电饭煲 / 蒸锅。
- 推荐做法：蒸、煮、炒、煎、炖、凉拌、破壁（米糊/豆浆/果蔬汁）、空气炸锅烘烤。

## 四、早餐（每天）
- **"三件套"硬要求**：每份早餐必须同时含 **主食 + 蛋 + 饮品**。
  - 主食：面 / 饭 / 馒头 / 饼 / 包 / 吐司 / 饭团 / 粥 等（如黄金馒头片、葱油拌面）。
  - 蛋：各种蛋、各种做法（水煮 / 煎 / 蒸蛋羹 / 厚蛋烧 / 荷包 / 卤蛋 / 溏心 / 鹌鹑蛋…）。
  - 饮品：牛奶 / 豆浆 / 米糊 / 果蔬汁 / 糊露 等。
- **一个月不重样**（早餐食谱池 ≥31，当前 34）。
- 晨间 ≤30 分钟；**前一晚可预备**（night_prep）。
- **儿童发育营养**：优质蛋白（蛋/奶/鱼/虾/牛肉）、钙（奶/黑芝麻/虾）、DHA（三文鱼/核桃）、粗粮膳食纤维。

## 五、节假日 / 周末正餐（午餐、晚餐）
- 每餐 **2–3 道菜**（通常 2–3）。
- **荤素搭配**：主菜含荤（肉/鱼/鸡/虾），加菜配素。
- **粗细搭配**：细粮主食（米/面）+ 至少 1 道粗粮（玉米/红薯/紫薯/南瓜/山药/杂粮糙米）。
  - 生成器实现：`generate_month.py` 的晚餐加菜 `prefer_category="粗粮"`，保证每个全餐日有粗粮。
- 必有主食；汤可选。
- 工作日只吃早餐（午晚在外/不安排）。

## 六、食材经济（减少浪费）
- 蔬菜/肉通常 **500g 起买**（最小购买量）。
- **某天用不完的食材，优先安排接下来几天继续消耗**。
  - 生成器实现：`pick_recipes` / `pick_side` 按 `ingredient_tags` 做**跨天食材聚类**，相邻几天优先复用同类食材。

## 七、目录与生成
- `recipes/breakfast|lunch|dinner|side|special/*.yaml`（special 为节日特供如粽子，不进日常轮换）。
- 加菜按 `category` 分：素菜 / 汤 / 凉菜 / **粗粮**。
- 生成：`python3 scripts/generate_month.py --year YYYY --month M`（只写 ≥今天的每日卡片）。
- 推送：`bash scripts/run_daily.sh` 或对话里说"食谱"；`notify_daily.py` 按当天日期读 `plans/daily/*.md`（北京 18:00 前=今日、之后=明日）。
