# 复现率Agent — 活文档

> 最后更新：2026-06-15 | 来源：复现率Agent.md + Q2 Wiki JSON W5–W12

## 背景与目标

用 Agent 替代人工复现率判定，覆盖 SIL 场景 12 个专项。验收标准：单专项人机一致率 ≥ 85%。  
Q2 目标：12 专项中至少 6 项验收上线；单 case 费用约 0.02–0.12 元；AI 可覆盖 28.66% 复现专项。

## 系统功能设计（来源：闭环复现分析Agent开发计划 K9eIw7MG）

### 使用链路

对话窗口输入 Job ID + 问题类别 → 提交任务 → Agent 执行 → 结果推送至用户（CSV 报告 + HTML 查看器）

1. 加减速顿挫
2. 导航变道晚 / 走错车道
3. 道内画龙
4. 变道 / 绕行 / 分合流 / 主辅路摆动
5. 进逆向车道
6. 不加速 / 加速慢（跑不到限速）
7. 红绿灯相关问题（闯红灯 / 冲黄灯）
8. 不减速 / 制动不足（撞动态车 / VRU）
9. 异常减速
10. 压线
11. 不居中 / 贴边
12. 撞路沿 / 障碍物

## AI改进方案（来源：复现Agent_AI改进 QV6uw07T）

### 5-27 讨论

- **DDS 时间线呈现**：标注 ①原始问题时间（暂无法获取）②triggertime ③接管时间 ④仿真车与实车分离点 — 后三项已实现 ✅
- **提示词关键因子**：红绿灯里参考了过多路径/角度，需增加 AI 合理判断轨迹的提示词 [ ]
- **Jerk 视图统一**：实车与仿真车曲线图统一横纵坐标，叠加一张视图 ✅

### 5-28 讨论

- **顿挫&摆动**：AI 审视范围聚焦问题 ✅；S/R 分离过大/接管后不做判别 ✅；比较次数而非单次 ✅
- **不加速**：看原始速度（差值有干扰）✅；聚焦问题范围（末端速度不合理）✅；S/R 速度差 ≤5km/h 算复现 ✅
- **制动不足**：过滤威胁距离异常波动 ✅；修正减速最小值 ✅；提示词有干扰待解决 [ ]
- **异常减速**：过滤无效合理减速 ✅；增加 ST 图（借鉴红绿灯视图）✅
- **压线**：虚实线和虚线检测 ✅；不稳定线型引导人工核实 ✅

### 5-29 讨论

- **撞路沿**：聚焦问题审视范围 [ ]；识别 trigger time 无效 case ✅；放大 BEV 快照 ✅
- **不居中**：居中阈值 ±0.3m 对比实车 ✅；车身分离较远不做判断 ✅；排除无效 case（轻微偏移/排队场景）✅

## 关键技术卡点

1. **顿挫专项**：低置信度 case 约 70%，仍未达 85% 目标；速度曲线优化后 76%，空间有限
2. **仿真/实车分离**：S/R 分离后因果颠倒导致误报，根因已知但需排期修复
3. **危险变道/绕行**：AI 聚焦距离而非趋势，7 case 全判不好，难度最大，放最后处理
