> 最后更新：2026-06-15 | 来源：TopDiff-Agent.md + Q2 Wiki JSON W11

## 背景与目标

闭环 Diff 分析，自动定位变道晚等问题的根因。通过对比 A/B 两版本轨迹 Top 指标差异，定向找出导致变道晚的关键因子。  
是复现率 Agent 的上游能力，目标定位准后再由复现率 Agent 判断是否复现。

TopDiff Agent 通过状态机驱动的串行专家工作流（4 节点），对 A/B 两版本轨迹的 Top 指标差异进行定向比对，定位导致变道晚的关键因子。作为复现率 Agent 的上游能力，其准确性直接影响下游复现判定质量。当前训练集准确率 6/7（约 86%），Top10 差异 case 中约 7 个有效（70%），但整体泛化准确率仍偏低。

1. **聚合函数**：当前路口场景误判多，单帧前后差异也导致误报，需设计更鲁棒的聚合策略
2. **旁车感知缺失**：绕行场景需要旁车信息辅助判断，当前未集成
3. **箭头/导航信息缺失**：期望加入车道箭头信息，结合高亮判断是否在正确车道，当前尚未实现 [来源：JSON W11]
