## AVM链路技术方案

（来源：avm链路-鱼眼 WDfGwUa0）

### 数据格式（calib.json cam9 关键字段）

**cam9（AVM前视鱼眼）配置**：

- 位置：`avm_front`，相机型号：ISX031，分辨率：1536×1920，hfov：28°
- 投影模型：**Mei 全方位相机模型**（`opencv_contrib_4.5.5_omnidir`）
  - `Xi = 0.9820293`（全方位参数，决定鱼眼畸变程度）
  - `D = [-0.2827, 0.0660, -0.00901, -0.00117, -0.000305]`（顺序：k1, k2, p1, p2, k3）
  - `K`（3×3）：fx=1125.52, fy=1124.66, cx=960.14, cy=770.37
- 外参附加字段：`H_global_SV2D`（3×3 单应矩阵，用于 AVM 俯视图拼接）

**与普通相机（cam0等）的区别**：

| 参数项 | AVM cam9（Mei模型） | 普通相机（cam0，针孔模型） |
|-------|------------------|------------------------|
| 投影模型 | Mei全方位，含 Xi=0.982 | 标准 OpenCV 针孔，无 Xi |
| 畸变参数 | D[5]: [k1,k2,p1,p2,k3] | k1~k6, p1, p2 分离存储 |
| 外参附加 | H_global_SV2D（俯视拼接） | 无 |
| 分辨率 | 1536×1920 | 2160×3840（cam0前窄） |
| is_avm | true | 无此字段 |

### 内参参数映射

cam9 参数向其他相机参数体系的映射：
- `focal_length ≈ (fx+fy)/2 = 1125.09`
- `cx = K[0][2] = 960.14`，`cy = K[1][2] = 770.37`
- `D[0]=k1, D[1]=k2, D[2]=p1, D[3]=p2`

### gsplat 参数 → cam9 Mei 模型参数映射（来源：sheet 7Fw2v1）

| gsplat参数 | cam9 Mei模型参数 | 值 | 说明 |
|----------|--------------|-----|------|
| Ks[0,0] (fx) | K[0][0] | 1125.51843 | x方向焦距 |
| Ks[1,1] (fy) | K[1][1] | 1124.66248 | y方向焦距 |
| Ks[0,2] (cx) | K[0][2] | 960.144958 | 主点x坐标 |
| Ks[1,2] (cy) | K[1][2] | 770.36554 | 主点y坐标 |
| radial_coeffs[0] (k1) | D[0] | -0.282716662 | 径向畸变1 |
| radial_coeffs[1] (k2) | D[1] | 0.065990947 | 径向畸变2 |
| radial_coeffs[2] (k3) | D[4] | -0.000304562 | 径向畸变3 |
| radial_coeffs[3] (k4) | ❌ 无 | 0 | 高阶径向畸变（建议设为0） |
| tangential_coeffs[0] (p1) | D[2] | -0.009012702 | 切向畸变1 |
| tangential_coeffs[1] (p2) | D[3] | -0.001166816 | 切向畸变2 |
| Xi参数 | 0.982029319 | ⚠️ gsplat不支持 | Mei模型特有参数 |

### camera_ray 与 image_point 函数关系（来源：sheet MP6Z4h）

| 函数 | 方向 | 使用场景 |
|------|------|--------|
| camera_ray → image_point | 3D→2D | 训练/渲染（高频调用） |
| image_point → camera_ray | 2D→3D | 仅 with_eval3d=True 射线追踪 |

### 鱼眼畸变处理方法

- **核心难点**：gsplat 内置 fisheye 模型不支持 Mei 模型中的 `Xi` 参数（0.982），直接映射存在精度损失
- **解决方案**：改造 Gisplan 代码，**新增 Mei 相机投影**，将 Xi 参数注入渲染管线
- **效果**：gsplat 代码已支持 Mei 模型，可接纳更多 AVM 相机（不只 cam9）

### 链路处理步骤

1. 解析 `calib.json` cam9 字段，提取 Mei 模型参数（Xi, D, K, H_global_SV2D）
2. 改造 Gisplan 渲染管线，注入 Mei 投影（替代原 OpenCV fisheye）
3. 用 H_global_SV2D 单应矩阵支持 AVM 俯视图（鸟瞰）拼接
4. 添加车身 Mask（Cam2 近地面被车身遮挡，需屏蔽无效区域）
5. 处理天空边界（Cam2 Sky 贴图边界问题）
