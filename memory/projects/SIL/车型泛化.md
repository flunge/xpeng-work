# 车型泛化验证

## 目标
A车型3DGS场景通过改变外参迁移到B/C车型→原位渲染+Difix修图→开环评测，验证"跨车型趋势一致性"。最终支撑630多车型发版验证（GIC刘先明提出）。

## 核心挑战
- ref图几何干扰：换车型后ref图会带偏3DGS几何（OOD问题）
- 车身修复：去掉ref图后车身被Diffusion修回原车型
- 优先级管控：3DGS生产缺乏优先级，高优任务被其他任务挤占
