# 复现率Agent

## 目标
用Agent替代人工复现率判定。12专项中6项验收上线（85%标准），当前4项已上线。
单case费用~0.02-0.12元。AI可覆盖28.66%复现专项，若顿挫+道内画龙收尾可达更高覆盖。

## 核心转变
**之前**：复现率人工review — 人力紧缺，一直没有系统性结论
**现在**：Agent自动化判断 — 12专项1024条输出(6/9)，4项上线，2项接近
**Gating数据集**：综合复现率67% → 剔除无效+prompt问题→87%（人机一致率74%）
