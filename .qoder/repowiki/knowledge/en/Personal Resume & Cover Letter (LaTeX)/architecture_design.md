Single-document LaTeX project rooted at `personal/resume/main.tex`, which acts as the document entry point: it selects the `moderncv` class with `classic` style and `orange` color, configures UTF-8/CJK encoding via `CJKutf8`, sets geometry and font packages, declares personal metadata (`\name`, `\title`, `\address`, `\phone`, `\email`, `\homepage`, `\photo`), then composes sections directly or by `\input`-ing modular fragments — `experienceCut.tex` (current resume experience), `publicationCut.tex` (selected publications), and an optional `coverletter.tex`. The directory ships a self-contained copy of the `moderncv` distribution (`moderncv.cls`, `moderncvcolor*.sty`, `moderncvstyle*.sty`, `moderncvcompatibility.sty`, `moderncvicons*.sty`) plus helper packages (`collection.sty`, `tweaklist.sty`) so the document can be compiled standalone without system-wide TeXLive dependencies. Images referenced by name (`photolk.jpg`, `picture.png`) live alongside the sources. Build target is `pdflatex main.tex` (or xelatex/lualatex) producing `main.pdf`.
