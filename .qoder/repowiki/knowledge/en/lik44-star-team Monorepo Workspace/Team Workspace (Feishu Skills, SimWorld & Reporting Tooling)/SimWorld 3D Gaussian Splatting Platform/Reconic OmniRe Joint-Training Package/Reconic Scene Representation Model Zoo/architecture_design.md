The package is a flat collection of interchangeable building blocks rather than a single monolithic model:
- `gaussians/` provides multiple splatting backends (Vanilla3DGS, PVG, DeformGS, Scaffold) that all consume the same per-gaussian xyz/opacity/scale/quaternion/DC-feature tensors and share rendering helpers.
- `nodes/` supplies rigid, deformable, and SMPL-based node representations; `human_body.py` wraps SMPLLayer with extra joints and a `VoxelDeformer` (from `modules.py`) for LBS weight interpolation, which is reused by deformable nodes.
- `generative_models/` sits on top as an optional post-processing stage (Difix / SD1.5 UNet) that takes rendered novel views and repairs them, optionally guided by sky masks.
- Cross-cutting components in the root (`modules.py`, `appearance_embedding.py`, `fourier_utils.py`, `losses.py`, `video_utils.py`) are imported by both gaussians and nodes: sinusoidal/Fourier positional encoders, `AppearanceEmbeddingModel`, `DeformNetwork`/`ConditionalDeformNetwork`, `SkyModel`/`EnvLight`, `CameraOptModule`, `AffineTransform`, and `Conv1DMLP`/`PositionalEncoding`. All modules follow a uniform registration pattern via a `class_prefix = class_name + "#"` attribute and expose `get_param_groups()` returning `{prefix + "all": self.parameters()}` so optimizers can group parameters consistently across subpackages.
