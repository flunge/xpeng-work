Requires external binaries on PATH: `ffmpeg` (with libx265) and `ossutil` configured with the hardcoded Wulanchabu endpoint and access keys in `dds_downloader.py`. Runs are driven by argparse flags on each script's `__main__`; `image_3dgs_to_dds.py` expects a directory layout of `<base_dir>/simulator_render/redistort_rgb_harmonized/<camN>/<timestamp>.png` and writes outputs under `<base_dir>/simulator_render/{hevc_videos,image_videos,origin_hevc_videos}`.
