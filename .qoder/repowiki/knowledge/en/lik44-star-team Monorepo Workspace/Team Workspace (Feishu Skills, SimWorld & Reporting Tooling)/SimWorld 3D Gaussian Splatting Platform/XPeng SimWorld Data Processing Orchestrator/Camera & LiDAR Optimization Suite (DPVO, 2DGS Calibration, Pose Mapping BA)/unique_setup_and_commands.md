Build environment is provided by `optimization/Dockerfile` (internal `fuyao` base). Inside the container, install steps are: `pip install -r camopt/requirements_campose.txt && pip install camopt`, `python lidaropt/lidar2cam_opt/submodules/diff-surfel-rasterization/setup.py install`, plus `torch-scatter` from the PyG CUDA wheel index. The end-to-end run is `bash lidaropt/pipeline_lidar2cam.sh`, which expects a KITTI-style dataset at `$DATA_ROOT` with per-camera dynamic/road masks and writes results back into `pcd_cvt_0/`.
