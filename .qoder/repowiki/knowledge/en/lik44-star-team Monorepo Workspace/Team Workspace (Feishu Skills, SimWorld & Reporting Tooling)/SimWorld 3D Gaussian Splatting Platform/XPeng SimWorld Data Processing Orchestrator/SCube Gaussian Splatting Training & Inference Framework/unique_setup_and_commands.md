Training is launched via `python train.py --model gsm --hyper configs/train/gsm/gsm.yaml [--resume] [--logger_type tb|wandb|none]`; inference via `python gsm_inference.py --ckpt_path <path> --output_root <dir>` which writes per-batch RGB gaussians as `.splat` files plus a merged `gsm_bkgd_init.ply`. The `ext/common/` CUDA sources are compiled on first import by `ext.__init__` using `torch.utils.cpp_extension.load` with `-O2` flags.
