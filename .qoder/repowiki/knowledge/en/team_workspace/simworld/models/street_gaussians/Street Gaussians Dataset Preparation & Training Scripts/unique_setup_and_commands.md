Requires an installed `colmap` binary on PATH (or pass `--colmap_executable` to `convert.py`) and ImageMagick `magick` for image pyramids; Waymo/KITTI pipelines expect data laid out under `cfg.source_path` with `extrinsics/`, `intrinsics/`, and precomputed masks; run dataset prep via `python <script> --source_path ...` then training via `bash train_waymo_example.sh` which loops over scenes defined in each shell file.
