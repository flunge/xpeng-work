Build from inside `gaussian-splatting/` after cloning recursively: `conda env create --file environment.yml && conda activate gaussian_splatting`, then `pip install submodules/diff-gaussian-rasterization submodules/simple-knn submodules/fused-ssim`. Training: `python train.py -s <COLMAP_or_NeRF_Synthetic_path> [-m output/<run>] [--eval] [--optimizer_type sparse_adam]`. Rendering: `python render.py -m <model_path> [--iteration N] [--skip_train/--skip_test]`. Full benchmark: `python full_eval.py -m360 <dir> -tat <dir> -db <dir>`.
