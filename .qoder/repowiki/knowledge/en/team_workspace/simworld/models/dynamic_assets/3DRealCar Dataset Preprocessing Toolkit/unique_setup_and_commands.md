Requires `apt-get install colmap` and `pip install colorama plyfile open3d kornia tqdm imageio imageio[ffmpeg] opencv-python supervision segment_anything submodules/GroundingDINO`, plus downloading `sam_vit_h_4b8939.pth` into `resources/models/`. The canonical run is `./bash/pipeline.sh <DATASET_NAME> <stage> <EXP_NAME> <dataset_dir> <codebase_dir>` where `<stage>` ∈ {`dataset`,`segmentation`,`pcd_clean`,`pcd_standard`,`pcd_rescale`}; `./bash/run.sh` loops over all stages sequentially. Each stage is guarded by marker files (`.dataset`, `.segmentation`, `.processed`) so re-runs skip completed steps.
