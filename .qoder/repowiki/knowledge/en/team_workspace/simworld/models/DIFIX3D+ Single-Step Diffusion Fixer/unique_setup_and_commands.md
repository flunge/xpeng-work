Install via `pip install -r requirements.txt`. Training is launched through `accelerate launch src/train_difix.py` (single GPU) or `--multi_gpu --num_processes N` (multi-GPU); the `deploy/run_train_difix.sh` and `deploy/deploy_difix_ngpu.bash` wrappers set `NUM_NODES`/`NUM_GPUS` and pass all hyperparameters from `configs/*.yaml`. Inference defaults to `num_inference_steps=1, timesteps=[199], guidance_scale=0.0` (single-step denoising).
